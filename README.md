# Landing page

This is just a simple landing page for a fictional company named Verdant Studio, who specializes in garden decorations.

This project was made using Haskell, HTML and CSS, mainly to get to know how to code in Haskell

## Sections
You have "Services", "Journal" and "Contact", with their respective nav buttons.

On services, you'll find a simple explanation of what this company can do, depending on what's required from the customer.

On Journal you'll find fictional journals that I was asked to invent for this site.

On contact you'll find a form to complete with your name, email and project idea, the send button won't work though, since it is just for learning haskell.
