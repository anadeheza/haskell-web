# My Writings
This is just a simple personal blogs website to post some of the stories I have written quite some time ago.

This project was made using Haskell, HTML, CSS, and JavaScript, mainly to get to know how to code in Haskell

![pagina](image.png)

## Sections
You have "Journal", "About" and "Quiz" sections, with their respective nav buttons.

On "Journal" you'll find some fictional stories that me and my sister have written a few years ago.
<br>
In each story, you can leave a message as a "margin note"

On "About" you'll find three cards that say a little bit about me as an amateur writer.

On "Quiz" you'll be able to take a multiple-choice to get to know what type of person you are and a personal recommendation of a book for you to read

## Difficulties
This year, i learnt to code in haskell at school, as a project, we were asked to build a website with a haskell backend

It took me way too long to get to understand haskell so it took me an insane amount of hours of coding for the type of project i've made since it is just a simple page but it has a lot of trial an error behind it, it was a real headache but i am glad i ended up understanding the language and it works just fine
